# TaskTracker: Intelligent Design Meets The File Organizer

**Nonhierarchical File Manager (developed in Visual Basic 6 between 2003 and 2007)**

Visit [TaskTracker](http://tasktracker.wordwisesolutions.com) site.

Compiles in Visual Basic 6, Service Pack 6

Originally designed for Windows XP/2000/2003/Vista/7

It has continued to work on systems up to and including earlier versions of Windows 10 with some degrading of the UI. 
However, in the current version of Windows 10 (and presumably 11), a Tablet PC check causes an error that shuts it down immediately. 
I have rectified that by commenting out CheckOS (line 380 in modBase.bas).

Future work: I have none planned.
